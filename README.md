## Deployment

These are standalone HTML files with no build step required. Push them to your repository:

```bash
git add .
git commit -m "Add 15 Cubical Golfer affiliate pages"
git push origin main
```

If uploading via GitHub web UI, upload each file individually. Avoid drag-and-drop of folders with bracket filenames (an Astro issue, not present in these pages).
